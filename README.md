# GalaxyNote10LiteROM
Custom Roms For Galaxy Note 10 lite


The Samsung Galaxy Note 10 Lite is a versatile smartphone that brings the core Note experience to a more affordable package. It pairs a large, vibrant display with the signature S Pen for precise input, and offers solid daily performance, a dependable battery, and capable cameras. For many users it’s an excellent balance of productivity features and value.

This repository will host custom ROMs for the Galaxy Note 10 Lite. You’ll find community-built firmware here focused on extended device support, performance improvements, additional features, privacy enhancements, and customization options not available in the stock software. Each ROM release will include clear installation instructions, changelogs, and compatibility notes to help you choose the right build and install it safely.

Please make sure to back up your data before installing any custom firmware, verify your exact device model, and follow the provided instructions for unlocking the bootloader and using a custom recovery. Contributions, bug reports, and pull requests are welcome — if you want to help maintain builds, documentation, or device-specific fixes, open an issue or a PR and we’ll coordinate.
